# 孙知羽（孙静）· 内容规划方案

> 本方案只规划纯内容（信任线），不包含带货内容。带货脚本另做专项规划。

---

## 一、人设定位

### 一句话

**清华法学 + 美护行业20年内部人，用拆解问题的思维方式帮宝妈把亲密关系和变美这两件事想清楚。**

### 凭什么听她的——三个支撑点

**支撑点1：清华的思维方式——拆解问题的底层工具**

不是用来炫耀的学历，是真实在用的工具。怎么拆解一个问题、怎么找到有效信息、怎么在一堆观点里判断哪个靠谱——这套方法是清华给她的。

**支撑点2：20年行业积累——从里面说行业的事**

从欧莱雅到大厂，产品、营销、品牌、供应链全链路做过。知道品牌方怎么定价、成分怎么包装、营销怎么制造焦虑。

**支撑点3：使命感——不是为了流量，是因为这批人值得更好的信息**

工薪家庭出身，她比谁都懂：每一分力气、每一分钱、每一分情绪，都不应该浪费在错误的地方。

---

## 二、核心人群

以**宝妈**为核心入口。产后脱发是把她们带进来的钩子，亲密关系和变美内容是让她们留下的理由，头发科普是建立品类信任的关键。

- 年龄：28-42岁
- 身份：宝妈为主（产后1-5年最集中），外扩到有头发困扰的女性
- 收入：中产为主（有消费力但在意性价比）
- 城市：一二线 + 强三线

**核心痛点：**
1. 产后脱发/头发焦虑
2. 亲密关系里的堵（丧偶式育儿、讲不通道理、出轨）
3. 当妈以后的自我消失
4. 变美焦虑（不知道什么真有用）
5. 信息过载

## 三、内容体系

### 总览

```
孙知羽纯内容
├── 泛娱乐线一：「用清华思维经营亲密关系」  ← 宝妈最在意的情感问题（引流）
├── 泛娱乐线二：「用清华思维变美」          ← 变美全域拆解（引流）
└── 垂类线：「清华思维养头发」              ← 头发头皮科普（精准转化）
```

> 三条线服务同一个人群。泛娱乐线把宝妈吸引进来、建立"这个人会帮我想清楚问题"的认知；垂类线把泛流量转化成"在头发这件事上信她"的精准用户，为自有头发品牌打地基。

---

### 泛娱乐线一：「用清华思维经营亲密关系」

**定位：第一大人设支柱，每周至少1条。**

一个具体的关系难题，把情绪问题翻译成可拆解的决策/结构问题，给出判断标准或行动步骤。

**内容公式：**

```
一个具体的人生难题
→ 把情绪问题翻译成决策问题
→ 给出拆解步骤或判断标准
```

**调性：** 不是情感博主的"抱抱你"，也不是说教的"你应该"。是"先别哭，咱把这事拆开看"——冷静、有逻辑、但不冷漠。

**选题库（20条）：**

**婚姻关系**
1. "男人出轨到底该不该离？先问自己这三个问题"
2. "老公什么都不管，我是该吵还是该离？先搞清楚你真正要的是什么"
3. "'丧偶式育儿'不是段子，是一个可以解的结构问题"
4. "婚姻里最累的不是吵架，是讲不通道理——怎么办"
5. "婆婆不帮带娃我该不该生气？这不是情绪题，是预期管理题"
6. "你以为'为了孩子不离婚'是爱？其实对孩子伤害最大"
7. "和老公吵架永远赢不了？不是你的问题，是沟通结构的问题"
8. "婚姻里的'冷暴力'为什么比吵架还可怕——用逻辑拆给你看"

**育儿关系**
9. "孩子成绩下滑我该怎么办？别急着报班，先做这一步"
10. "要不要'鸡娃'这道题——我的判断标准就一个"
11. "我们这代妈妈最容易犯的一个错，是把自己的焦虑转嫁给孩子"
12. "孩子不听话？先搞清楚他不听的到底是哪句话"

**自我关系**
13. "全职妈妈该不该重返职场？把它当成一道决策题来算"
14. "当妈以后焦虑的根源不在孩子——我后来想明白了"
15. "40岁了还有没有可能重新开始——不是鸡汤，是算账"
16. "为什么当了妈以后越来越不认识自己——问题出在这里"
17. "产后掉下去的不只是头发——怎么一件一件捡回来"

**人生决策**
18. "清华法学生转行做美妆，所有人说我疯了——我是怎么算这笔账的"
19. "40岁裸辞不是勇敢，是算过账的——我的决策框架分享给你"
20. "工薪家庭出身，我是怎么学会不再为钱焦虑的"

### 泛娱乐线二：「用清华思维变美」

**定位：第二大人设支柱，每周1条。**

一个常见的变美困惑/焦虑，用行业内部逻辑或研究数据拆解真相，给出清晰判断和可执行的方法。

**内容公式：**

```
一个常见的变美困惑/焦虑
→ 用行业内部逻辑或研究数据拆解真相
→ 给出清晰判断和可执行的方法
```

**调性：** 20年行业内部人视角 + 清华拆解方法，不种草不贩卖焦虑。"我在里面做过，所以我知道哪些是真的、哪些是收你智商税。"

**选题库（20条）：**

**变美优先级（打破常识）**
1. "30岁以后变美最该花钱的地方，不是精华不是面膜"
2. "变美这件事，先想明白顺序——顺序错了花再多钱都没用"
3. "变美先从变不累开始——睡眠和气血是所有护肤品都替代不了的"

**护肤真相**
4. "一瓶300块的护肤品，成本到底是多少——我在大厂做过定价"
5. "你以为贵的护肤品一定好？定价逻辑跟成本没关系"
6. "'经临床验证'这四个字，背后到底验证了什么"
7. "同一个工厂出来的东西，大牌卖500白牌卖50——哪些钱值得花"
8. "'植物温和''无添加'这些词我越听越害怕"
9. "护肤品成分表有这个成分≠有效——品牌方玩的是这个把戏"

**医美与项目判断**
10. "医美什么值得做，什么是在收割焦虑——行业视角来说"
11. "热玛吉、超声炮、水光针——同一个目标几种方法，怎么选"
12. "贵妇院做的和普通诊所做的，差距到底在哪"

**身体与状态**
13. "压力会让你垮，这件事我用数据给你看"
14. "30岁以后，身材管理的逻辑和20岁完全不一样"
15. "你以为是黑眼圈，其实是身体在给你报警"

**产后身体重建（宝妈核心痛点）**
16. "产后脱发到底该不该治？什么时候治？在这行待了20年告诉你"
17. "产后皮肤变差、头发变少、整个人垮了——先搞清楚优先级"
18. "生完孩子后最不该省的一件事，不是护肤品"

**消费判断框架**
19. "判断一个护肤品值不值得买，看这三个维度"
20. "我做了20年美护，踩过最贵的三个坑——行业里的人也会被收割"

### 垂类线：「清华思维养头发」

**定位：垂直品类科普，直接服务于头发品牌。每周1条。**

用20年行业积累 + 清华拆解方法，围绕头发头皮做深度科普。不卖货、不种草，纯粹帮用户搞清楚头发这件事的底层逻辑。这条线的目标是：让关注孙知羽的人，在头发问题上只信她。

**内容公式：**

```
一个关于头发/头皮的常见困惑或错误认知
→ 用行业内部逻辑 + 科学数据拆解真相
→ 给出清晰的判断标准或可执行的方法
```

**调性：** 专业但说人话。不是皮肤科医生的术语课，是"在这行做了20年的人跟你讲讲头发这件事到底是怎么回事"。有行业内部视角（品牌怎么包装概念、哪些成分是营销造的），也有科学视角（脱发的真实机制、头皮的基本逻辑）。

**选题库（15条）：**

**脱发认知（破除误区）**
1. "脱发这件事，90%的人第一步就走错了——换洗发水是最没用的一步"
2. "产后脱发和雄激素脱发完全不是一回事——搞清楚你是哪种再花钱"
3. "你以为产后脱发会自己好？80%的人没等到那一天"
4. "掉发量多少算正常？网上说的'每天100根'根本不准——正确的判断方法是这个"
5. "压力是怎么让你的头发变少的——从激素到毛囊，我把这条链拆给你看"

**头皮科学（建立正确认知）**
6. "头皮是皮肤，但你护肤那套逻辑用在头皮上全错了"
7. "头皮出油不是因为洗少了——越洗越油的真实原因"
8. "头皮屑和头皮干，很多人分不清，但处理方法完全相反"
9. "睡眠不够和头皮出油是同一个问题的两个结果——从数据出发来说这件事"
10. "你的头发细软，99%的人归因错了——和吃什么关系没那么大"

**产品与成分（行业内部人视角）**
11. "你以为'氨基酸洗发水'温和？这个概念是营销造出来的"
12. "'防脱洗发水'到底有没有用？在这行待了20年，我来说真话"
13. "防脱成分哪些有用哪些是智商税——我把市面上的成分过一遍"
14. "'无硅油更健康'这件事，是品牌讲的一个很漂亮的故事"
15. "为什么我做了20年美护，最后选择做头发品牌"

## 四、爆款规律

### 1. "清华思维"是天然钩子

"用清华思维"前缀自带停留力——用户好奇清华的人怎么想问题。但钩子只管进门，内容质量决定留存。

### 2. 宝妈痛点 + 反常识判断 = 最强组合

选题同时命中"宝妈正在经历的事"和"跟大众认知不一样的判断"，爆款概率最高。比如"你以为产后脱发会自己好？""丧偶式育儿不是段子，是结构问题"。

### 3. 开头钩子的常用模式

- **反常识钩子**："你以为……其实……"
- **行业内部人钩子**："在这行待了20年，我告诉你……"
- **拆解钩子**："这个问题，我用清华学的那套方法拆了一遍"
- **经历钩子**："40岁裸辞那天，我脑子里在想什么"

### 4. 什么类型选题容易爆

- 婚姻关系类（出轨、丧偶式育儿）——情绪共鸣 + 社交货币
- 行业揭秘类（定价真相、营销套路）——信息差
- 产后刚需类（脱发、皮肤、身材）——精准痛点
- 人生选择类（裸辞、转行）——故事感
