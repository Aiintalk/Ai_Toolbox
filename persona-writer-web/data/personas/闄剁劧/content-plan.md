# 陶然 · 人设 & 内容规划方案

---

## 一、人设定位

### 一句话

**真的花钱 · 真去试 · 真看数据 · 真敢说**

### 四个关键词

| 关键词 | 内涵 | 用户感知 |
|--------|------|----------|
| **真的花钱** | 赚的钱大半砸在抗衰和变美上，投入量级远超同行 | "她是真花了钱的" |
| **真去试** | 所有推荐过的东西，自己先用过 | "她替我试过了" |
| **真看数据** | 不凭感觉说好，必须拿检测数据说话 | "她说的有依据" |
| **真敢说** | 好就说好，不好就说不好，北京人净说大实话 | "她不骗我" |

### 凭什么听她的——两个支撑点

**支撑点1：花过的钱**

她不是医生不是专家，她的权威来自真金白银的消费经验。同样有钱的情况下，她真的愿意一个一个去体验，然后把结果公开出来。

- 干细胞：飞国外做了4次，花了大几百万，发现4次颜色都不一样
- 医美：存了100万准备全身爆改，做了一圈发现没用还留了痕迹
- 菌群移植：从2024年到现在做了5次
- 各种高端身体管理项目：持续在体验、持续在花钱

**支撑点2：测得到的数据**

"看得到的状态"太主观——谁都可以说自己变好了。陶然要做的是**用数据说话**。

配合即将推出的抗衰检测产品（唾液检测盒，可检测甲基化水平、NAD+含量、端粒长度等生物衰老指标），让陶然成为：**第一个把自己的抗衰数据公开给观众看的主播**。

- 定期公开自己的检测数据
- 用数据对比体验前后的变化
- 用数据替代"感觉好了"这种主观表达

**这两个点叠加就是陶然独一无二的壁垒**：别人只会说"感觉好了"，她拿得出钱花了多少和数据变了多少。

### 主页简介

```
40岁 · 三胎妈妈 · 北京

10年花了7位数的健康消费踩坑指南
用数据管身体 不凭感觉花钱

替你试过了 ✓
```

「十年减龄计划」从主页定位降级为内容栏目，不再写在简介中。

## 二、内容体系

### 总览

```
陶然内容
├── 纯内容（约50%）
│   ├── 「我有一个朋友」系列
│   ├── 「我替你试过了」系列
│   └── 「我想明白了」系列
│
└── 带货内容（约50%）
    围绕每周大型直播，打包为一套「直播内容组」
    ├── 深度种草（先破后立型）
    ├── 日常露出（素颜vlog/原相机型）
    ├── 溯源 vlog
    ├── 营销预热（锦鲤/福利预告）
    └── 机制释放
```

> 带货内容围绕每周大型直播排期，一周内会发多条，另做专项排期。本方案先规划纯内容部分，再给出带货内容的营销框架。

### 系列一：「我有一个朋友」

**定位：第一大人设支柱，每周至少1条。**

借身边朋友的真实故事，站在"过来人"的视角输出判断和观点。朋友大多是有钱有故事的女性，生活本身就有猎奇看点。陶然带着北京大姐的直率，对这些朋友的选择给出真实评价。

**内容公式：**

```
故事钩子（我有个朋友，她XX）
→ 朋友的经历/选择/踩坑（猎奇感+细节）
→ 陶然的判断和观点（我觉得她XX / 我当时就跟她说XX）
→ 落回一个普适的道理或建议
```

**为什么这个系列是核心：**

1. **猎奇**——白富美朋友的故事自带流量
2. **距离刚好**——不是遥远的名人，是"身边的朋友"，可信度高
3. **人设加分**——陶然对富婆朋友直言不讳的姿态，体现了"真实"和"较真"
4. **不可复制**——只有陶然能讲，因为她真的有这些朋友

**选题库（20条）：**

1. "我有个朋友，花了50万做医美，最后跟我说了一句话"
2. "我有个朋友，在美容院被忽悠了80万的干细胞"
3. "我有个朋友，每年花30万养生，身体反而越来越差"
4. "我有个朋友，年入千万但焦虑到失眠，她后来做了一件事"
5. "我有个朋友，白富美，40岁了还在为体重焦虑"
6. "我有个朋友，离婚的时候什么都没要，现在活得最好"
7. "我有个朋友，全家人劝她别生三胎，她还是生了"
8. "我有个朋友，老公出轨后她做了一个所有人都没想到的决定"
9. "我有个朋友，孩子送进最好的学校，但她说最后悔的是这件事"
10. "我有个朋友，从月薪3000到年入8位数，靠的不是能力"
11. "我有个朋友，是我见过最会花钱的女人"
12. "我有个朋友，40岁卖了所有奢侈品，理由让我沉默了"
13. "我有个朋友，做了全脸整形，回来我第一句话就是你傻了吗"
14. "我有个朋友，嫁了有钱人但过得最苦"
15. "我有个朋友，为了备孕花了20万请营养师"
16. "我有个朋友，生完孩子老了10岁，但她做了一件事"
17. "我有个朋友，是我见过最清醒的全职太太"
18. "我有个朋友，被闺蜜骗了200万，她的反应让我震惊"
19. "我有个朋友，40岁重新创业，所有人都说她疯了"
20. "我有个朋友，每次见面都比上次年轻，我终于问出了原因"

### 系列二：「我替你试过了」

**定位：第二大人设支柱，每周1条或每两周1条。**

陶然花自己的钱，替观众去体验那些贵的、好奇但不敢试的身体管理项目，真实记录体验过程和感受。

核心逻辑：**"这些东西你好奇但舍不得花钱试，我替你试了，值不值我跟你说实话。"**

**内容公式：**

```
项目介绍（这个东西是什么、多少钱）
→ 真实体验过程（全程记录）
→ 真实感受（值不值、体感如何、有没有坑）
→ 如果有检测数据，前后数据对比
```

**这个系列的价值：**

1. **天然猎奇**——10万一次的正骨、大几百万的干细胞、专业版菌群移植——这些项目本身就有看点
2. **消费权威**——花过的钱就是壁垒，别人写测评是看资料，她是真花了钱
3. **可以和检测数据结合**——体验前测一次，体验后测一次，用数据说话
4. **自然服务于带货**——体验完觉得好的，直播间可以卖；觉得不好的，做成避坑内容

**选题库（15条）：**

1. "10万块一次的小颜正骨，是什么体验"
2. "大几百万的干细胞，我做了4次告诉你真相"
3. "专业版菌群移植，和普通版到底差在哪"
4. "一次5万的全身基因检测，值不值"
5. "传说中的日本精密体检，我飞去做了一次"
6. "韩国顶级皮肤科的人胶原焕肤，折合1万多一次"
7. "3000块一节的普拉提私教，和免费跟练有什么区别"
8. "花了8万做了一套口腔正畸，我的建议是"
9. "一个月2万的月子餐 vs 自己请阿姨做，差距在哪"
10. "传说中贵妇才做的淋巴引流，800一次值不值"
11. "在三甲做了一次完整的肠道菌群检测，结果出乎意料"
12. "花了2万做了一次全面营养素检测，发现自己缺了这些"
13. "试了一个月的高压氧舱，真实体感是这样的"
14. "中医调理3个月 vs 西医方案3个月，我的身体数据对比"

#### 长线子系列：「造一个抗衰检测盒子」

这是「我替你试过了」下的一条持续更新线，记录陶然从零开始做一款抗衰检测产品的全过程。不是等产品做好了再发，而是**把做产品的过程本身变成内容**。

为什么这条线重要：
1. **粉丝参与感**——不是"我做好了你买"，是"我们一起做"
2. **天然信任**——全过程透明，用户看着产品长出来的，上线时不需要种草
3. **持续更新**——每一步都是内容，不会断更

**内容节点（按推进顺序）：**

1. "我想做一个东西：一个唾液就能测衰老速度的盒子"（起念）
2. "今天去探访了第一家检测机构"（溯源）
3. "和专家聊了2小时，终于搞明白甲基化是什么"（拜访专家）
4. "你们最想测哪些数据？评论区告诉我"（粉丝共创）
5. "甲基化、NAD+、端粒长度——我选了这几个指标，原因是"（决策公开）
6. "今天去看了生产企业，跟你们说说里面什么样"（工厂溯源）
7. "样品出来了，我先拿自己测"（第一个用户是自己）
8. "我的检测结果出来了：我的生物年龄居然是XX岁"（数据公开）
9. "找了10个姐妹一起测，结果差异比想象的大"（小规模测试）
10. "这个盒子终于可以卖了，但我想先跟你们说说它不能做什么"（坦诚上线）

### 系列三：「我想明白了」

**定位：个人认知输出系列，每周1条。**

陶然用自己的经历做底，输出笃定的观点和可执行的方法论。不是"聊聊感悟"，是**"我想明白了，跟你们说说"**。带一点能量/轻玄学的调子，带一点犀利不怕得罪人的锋芒。这个系列不带任何产品。

**内容公式：**

```
钩子（一个反常识的观点 / 一句笃定的话 / 一个冲突场景）
→ 用自己的真实经历做支撑（故事带观点）
→ 给出具体方法论（2-3步可执行）
→ 笃定收束，不软着陆
```

**选题库（15条）：**

1. "40岁以后我只做三件事：打扮、学习、赚钱"
2. "离过一次婚的人，第二次选人只看一个标准"
3. "女人最贵的资产不是脸，是气血——我用3年验证的"
4. "受苦比解决问题轻松，这句话价值100万"
5. "我花了10年才学会一件事：不解释。方法就三步"
6. "远离认知低的人，我有三个判断标准"
7. "心想事成不是玄学，是我验证过的方法论"
8. "40岁的女人不需要安全感，需要底气——底气从哪来"
9. "嫁得好不如活得旺，旺的女人有这3个特征"
10. "每年花一笔钱做身体检测，这是我花过最值的钱"
11. "我从全职太太到年入8位数，靠的不是能力是这一点"
12. "40岁生三胎所有人都说我疯了，但我算得很清楚"
13. "女人的好运气不是等来的，我有3个养运气的方法"
14. "不要和没有结果的事情纠缠——我的断舍离不是扔东西"
15. "结局一定是好的，如果还不够好说明还没到结局"

### 更新频率

每周 3 条纯内容，分配到三个系列。带货内容围绕每周大型直播另做排期。

---

### 带货内容：完整营销周期框架

以下框架来自一次真实的完整营销周期（生命种子+胎一，约2周，20条短视频），已跑通数据验证。每次大型直播前，按此框架组织一套带货内容。

#### 八种内容类型

| 类型 | 作用 | 时长 | 说明 |
|------|------|------|------|
| **心智内容** | 价值观共鸣，建立情感连接 | 2-5分钟 | 用真实经历讲一个和产品价值观相关的故事，不直接卖货但建立"为什么做这件事"的动机。如：致父亲的一封信、健康气血是终极优越感 |
| **日常露出** | 展示真实状态，自然带出产品 | 15-60秒 | 随手拍、素颜出镜、旅行vlog，产品出现在生活场景里而不是正对镜头介绍。如：巴厘岛街边随手拍、到贵州了 |
| **溯源 vlog** | 建立品质信任 | 1-3分钟 | 去工厂/源头实地探访，展示生产线、资质、检测报告。从出发到抵达到参观，可以拆成多条。如：到喽→从源头替大家考察了一遍→带大家到源头 |
| **名人/专家背书** | 权威加持 | 1-3分钟 | 引用行业权威、名人关联，增加可信度。不是请人站台念词，是在真实场景中产生交集。如：布莱恩硅谷抗衰、梅耶马斯克对话 |
| **粉丝互动** | 制造参与感 | 30-60秒 | 征集试品官、评论区投票、粉丝共创。让用户从"被推销"变成"被邀请参与"。如：在线征集100位试品官 |
| **深度种草** | 核心转化内容 | 2-5分钟 | 先破后立型：先把品类的坑讲透、揭行业乱象，再用数据和经历推产品。这是GMV最高的内容类型。如：生命种子种草16.6w |
| **机制释放** | 临门一脚，促转化 | 30-90秒 | 直播前释放价格、赠品、限量信息，随手拍风格，简短有力。如：天啊（机制随手拍）、0门槛满赠 |
| **用户反馈** | 社会证明 | 15-30秒 | 快剪真实用户反馈，用量级说话（2万条反馈、99%好评率）。如：种子用户反馈快剪 |

#### 发布节奏（约2周一个周期）

```
第1阶段：心智铺垫（直播前10-7天）
├── 心智内容 × 1-2条（情感故事，建立价值观）
└── 日常露出 × 1-2条（随手拍，展示真实状态）

第2阶段：预热蓄水（直播前7-4天）
├── 溯源 vlog × 2-3条（从出发到实地，拆成多条发）
├── 名人/专家背书 × 1条
└── 粉丝互动 × 1条（征集试品官/投票）

第3阶段：深度种草（直播前3-1天）
├── 深度种草 × 1-2条（先破后立，核心转化内容）
└── 产品科普 × 1条（专家采访/成分解读）

第4阶段：机制释放 + 直播（直播当天及前1天）
├── 机制释放 × 1-2条（福利预告，简短有力）
├── 用户反馈 × 1条（快剪，社会证明）
└── 直播
```
