# 陶然 · 内容规划方案

> 本方案只规划纯内容（信任线），不包含带货内容。带货脚本另做专项规划。

---

## 一、人设定位

### 一句话

**花过几百万踩坑的超级消费者，用真金白银的教训帮高净值姐姐们在护肤和健康上少走弯路。**

### 凭什么听她的——两个支撑点

**支撑点1：超级消费者——花过的钱就是专业壁垒**

100万医美、几百万干细胞、眼周200多支薇旖美、空瓶无数。她不是在推荐产品，是在报告踩坑结果。用户信她，因为她花的比用户多。

**支撑点2：数据说话——不凭感觉，用检测结果**

不说"感觉皮肤变好了"，说"检测数据是多少"。用具体的时间线、使用量、复购记录、检测报告替代主观感受。

---

## 二、核心人群

以**高净值女性**为核心。30-50岁、有家庭、经济独立或家庭富裕，能接受客单价1万的消费。

- 年龄：30-50岁
- 身份：有家庭有孩子的成熟女性，经济独立
- 收入：高收入（客单价约1万）
- 城市：一二线为主

**核心痛点：**
1. 抗衰焦虑——皮肤和身体在走下坡路，市面上产品鱼龙混杂
2. 被收过智商税——花过钱踩过坑，对营销话术有免疫力，但仍然需要可信的信息源
3. 健康管理困惑——身体开始出信号，想做点什么但不知道从哪入手
4. 想要"过来人"而非"专家"——不想听学术语言，想听一个跟自己一样的人说大白话

## 三、内容体系

### 总览

```
陶然纯内容
├── 泛娱乐线一：「我想明白了」    ← 人生智慧，笃定输出（引流）
├── 泛娱乐线二：「我有个朋友」    ← 故事带观点，猎奇引流（引流）
├── 泛娱乐线三：「能量小妙招」    ← 生活化身体智慧（引流）
└── 垂类线：「体验xxx」           ← 超级消费者体验记录（精准转化）
```

> 泛娱乐三条线把不同兴趣的女性吸引进来——人生感悟、朋友故事、身体小妙招都是大众话题。垂类线「体验xxx」是陶然独一无二的壁垒：花真金白银替用户体验高端项目和产品，建立"在护肤和健康消费上信她"的品类信任，直接服务于直播带货的转化。

---

### 泛娱乐线一：「我想明白了」

**定位：第一大人设支柱，每周1条。**

陶然用自己的经历做底，输出笃定的观点和可执行的方法论。不是"聊聊感悟"，是"我想明白了，跟你们说说"。这个系列不带任何产品。

**内容公式：**

```
以前怎么想的
→ 经历了什么
→ 现在想明白了什么
```

**调性：** 不是鸡汤，是真的走过弯路之后的笃定。语气是"活明白了"不是"好惨"。

**选题库（10条）：**

1. "我想明白了，40岁女人最大的资产不是脸"
2. "28岁辞职那天，我妈以为我疯了——现在她不这么想了"
3. "离婚后我明白了一件事：从来没有'失去一切'这回事"
4. "我想明白了，所有让你焦虑的东西，都有人在靠它赚钱"
5. "40岁以后我只做三件事：打扮、学习、赚钱"
6. "离过一次婚的人，第二次选人只看一个标准"
7. "受苦比解决问题轻松，这句话价值100万"
8. "我花了10年才学会一件事：不解释。方法就三步"
9. "40岁的女人不需要安全感，需要底气——底气从哪来"
10. "40岁生三胎所有人都说我疯了，但我算得很清楚"

---

### 泛娱乐线二：「我有个朋友」

**定位：故事带观点系列，每周1条。**

借身边朋友的真实故事，站在"过来人"的视角输出判断和观点。朋友大多是有钱有故事的女性，生活本身就有猎奇看点。

**内容公式：**

```
朋友的真实故事开头
→ 从故事里长出一个判断/道理
→ 用自己的经验做验证
```

**调性：** 用别人的故事说自己想说的话，比"我认为"更有信服力。

**选题库（10条）：**

1. "我有个朋友，四年没断顿买一个东西——后来我去查了为什么"
2. "我闺蜜去做了热玛吉，回来跟我说了一句话"
3. "我有个朋友38岁离婚，现在活成了我最羡慕的样子"
4. "我闺蜜花了20万做全脸填充，三个月后她哭着来找我"
5. "我有个朋友，花了50万做医美，最后跟我说了一句话"
6. "我有个朋友，在美容院被忽悠了80万的干细胞"
7. "我有个朋友，年入千万但焦虑到失眠，她后来做了一件事"
8. "我有个朋友，老公出轨后她做了一个所有人都没想到的决定"
9. "我有个朋友，是我见过最清醒的全职太太"
10. "我有个朋友，每次见面都比上次年轻，我终于问出了原因"

---

### 泛娱乐线三：「能量小妙招」

**定位：生活化身体智慧系列，每两周1条。**

一个日常现象或身体信号，用气血、作息、情绪管理的角度解释，给一个马上能做的小方法。

**内容公式：**

```
一个日常现象/身体信号
→ 用气血、作息、情绪管理的角度解释
→ 给一个马上能做的小方法
```

**调性：** 生活化，不讲理论讲体感。不是玄学课，是"我自己试过有用的小习惯"。

**选题库：**

1. "我发现一个规律：心情好的那个月，皮肤一定好"
2. "40岁以后最该做的一件事不是护肤，是早睡"
3. "为什么有些人40岁看着像30岁——不是基因，是生活习惯"
4. "我每天花10分钟做的一件事，比任何精华都管用"

---

### 垂类线：「体验xxx」

**定位：垂直品类体验，直接服务于直播带货转化。每周至少1条。**

陶然花自己的钱，替观众去体验那些贵的、好奇但不敢试的项目和产品，真实记录体验过程和感受。踩坑的说踩坑，好东西也大方说好。这条线是陶然最核心的壁垒——别人只能说"感觉好了"，她拿得出花了多少钱和数据变了多少。

**内容公式：**

```
为什么去做 → 花了多少钱
→ 过程和真实感受
→ 结果（数据/效果）
→ 值不值得
```

**调性：** 纪录片感，踩坑的说踩坑，好东西也大方说好。"我替你们趟过了，好的坏的都告诉你。"

**选题库（12条）：**

**高端项目体验**
1. "体验100万全身医美：做完一圈，告诉你哪个有用哪个没用"
2. "体验干细胞回输：飞国外扎了4次，4次颜色都不一样"
3. "体验菌群移植：做了5次，说说到底值不值"
4. "10万块一次的小颜正骨，是什么体验"
5. "专业版菌群移植，和普通版到底差在哪"
6. "一次5万的全身基因检测，值不值"
7. "传说中的日本精密体检，我飞去做了一次"
8. "韩国顶级皮肤科的人胶原焕肤，折合1万多一次"

**产品与方案体验**
9. "试了一个月的高压氧舱，真实体感是这样的"
10. "我用了一年的东西，说说哪些是真的离不开了"
11. "中医调理3个月 vs 西医方案3个月，我的身体数据对比"
12. "38岁卵巢早衰，40岁自然怀孕——这三年我做了什么"

### 更新频率

每周 4 条纯内容：我想明白了 1 条 + 我有个朋友 1 条 + 体验xxx 至少 1 条 + 能量小妙招每两周 1 条轮换。

## 四、爆款规律

基于TOP10内容的共性分析：

### 1. 真实消费量级是最强钩子

"花了几百万""做了4次""用了一年"——具体的金额、次数、时间线比任何标题技巧都管用。数字越具体，可信度越高。

### 2. 先破后立是核心逻辑

前半段先把品类的坑、竞品的问题、行业乱象说透，再推自己的判断。用户信了"你说的坑"，才会信"你说的道理"。

### 3. 开头钩子的常用模式

- **金额钩子**："花了XX万做了XX，告诉你……"
- **反常识钩子**："你以为XX？其实……"
- **朋友故事钩子**："我有个朋友，她XX——"
- **笃定观点钩子**："我想明白了，XX"

### 4. 什么类型选题容易爆

- 高金额体验类（干细胞、医美、高端项目）——天然猎奇
- 朋友故事里有反转/冲突的——社交货币
- 人生转折类（离婚、裸辞、生三胎）——情绪共鸣
- 生活小妙招类——门槛低、易传播
