# Changelog

记录短视频字幕提取工具的版本更新历史。

---

## v1.0.0 - 2026-05-16

### Added
- 初始版本发布
- 单条抖音视频字幕提取（TikHub解析 + 阿里云ASR转写）
- Excel批量上传，自动提取字幕，支持进度实时展示
- 批量结果下载（5列Excel：视频标题、抖音链接、字幕内容、转换状态、失败原因）
- 上传并直接下载模式（完成后自动触发下载，无需手动操作）
- AI思维导图生成（基于字幕内容，运营视角六维分析）
- 思维导图前端缓存，同一视频无需重复调用AI
- 接入现有工具箱（Nginx代理 `/subtitle-extractor`，端口3012）
- 新增 `README.md`、`API_CONTRACT.md`、`CHANGELOG.md` 文档
