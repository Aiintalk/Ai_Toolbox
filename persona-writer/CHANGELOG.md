# Changelog

记录本项目的重要功能变更、问题修复和文档更新。

---

## v2.0.0 - 2026-04-08

### Added
- 初始版本，三步人设脚本仿写流程完整开发
- 支持多达人风格档案加载与切换
- 支持抖音视频解析（TikHub）与点赞量门槛校验
- 支持 AI 开头吸引力评估（可跳过）
- 支持对标脚本结构自动拆解
- 支持三种仿写方向选择（有想法 / 没想法 / AI 推荐选题）
- 支持多轮对话迭代修改脚本
- 支持向达人素材库添加 / 删除参考内容
- 新增 `README.md` 和 `CHANGELOG.md` 文档

### Notes
- 达人列表调用 `/material-library/api/personas`，需配合 material-library 服务部署
- ASR 接口已实现，但当前 UI 版本改为手动粘贴文案，转录接口处于备用状态
- `data/personas/` 目录需在部署后手动创建并维护达人档案
