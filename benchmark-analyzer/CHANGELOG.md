# Changelog

记录本项目的重要功能变更、问题修复和文档更新。

---

## v2.0.0 - 2026-04-08

### Added
- 初始版本，对标账号分析流程完整开发
- 支持抖音号 / 主页链接自动抓取账号数据（TikHub）
- 支持手动粘贴文案数据
- AI 流式生成人格档案与内容规划（`claude-sonnet-4-6`）
- 支持分 Tab 展示、一键复制、导出 Word 文档
- 历史记录自动保存与加载
- 新增 `README.md` 和 `CHANGELOG.md` 文档

### Notes
- 历史记录依赖本地文件系统（`data/` 目录），重启后需挂载持久化卷
- TikHub 翻页上限 200 条，超大账号数据抓取不完整
