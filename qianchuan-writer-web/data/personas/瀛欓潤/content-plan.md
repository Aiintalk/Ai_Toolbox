孙知羽（孙静）· 实战内容规划 v1.0

## 一、人设定位

### 一句话

清华出来的 · 行业真里做过 · 真的想明白了 · 真敢告诉你

### 四个关键词

| 关键词 | 内涵 | 用户感知 |
|---|---|---|
| 清华思维 | 不是学历标签，是真实在用的拆解工具 | "她想问题和别人不一样" |
| 20年行业内部人 | 从欧莱雅管培生到大厂全链路，从里面出来的 | "她是真懂行的" |
| 过来人 | 工薪家庭、婚姻、孩子、裸辞——和用户站在同一个地面上 | "她懂我" |
| 敢说真话 | 有立场，有判断，不模棱两可 | "她不骗我" |

### 凭什么听她的——两个支撑点

**支撑点1：她从那边走出来**

她不是消费者，也不是专家。她的权威来自一件事：她曾经在行业里真正做过这些事。

- 欧莱雅管培生起步，从最近的距离看清楚大品牌怎么运转
- 大厂全链路20年——产品、营销、品牌、供应链全走过
- 知道一款产品从配方到定价到营销的完整逻辑，因为她亲手做过
- 知道品牌怎么用成分制造焦虑，因为她开过那些会

这意味着：她说"这个东西是智商税"，不是消费者的直觉，是内部人的判断。这是绝大多数博主没有的东西。

**支撑点2：她主动放弃了**

放弃本身就是信任的背书。
大厂里有职级、有薪水、有稳定。她40岁裸辞，不是混不下去，是想清楚了：大厂服务的是品牌，她想服务的是被品牌收割的普通人。

这个方向的转换，比任何资历证书都有说服力——因为它意味着她说话不需要维护任何品牌利益。

这两点叠加就是她独一无二的壁垒：
别人只会说"这个成分很好"，她能告诉你这个成分是怎么被包装成卖点的，以及为什么她选择站在消费者这边说这句话。

### 主页简介

```
41岁 · 北京 · 清华本硕 · 一个5岁女儿的妈
欧莱雅管培生→大厂美护20年→40岁裸辞
我在行业里见过品牌怎么赚你的钱
现在出来告诉你
替你想清楚了 ✓
```

## 二、内容体系

### 总览

```
孙知羽内容
├── 纯内容（约60%）
│   ├── 「我在那边做过」系列   ← 行业内部人视角，建立专业壁垒
│   ├── 「这道题我答出来了」系列 ← 中年女性全域焦虑破局，建立情感连接
│   └── 「清华教我这么想」系列  ← 思维工具输出，建立认知差异化
│
└── 带货内容（约40%）
    围绕头发品类深耕及选品推荐，打包为一套「选品内容组」
    ├── 先破后立型种草（行业逻辑拆解→推荐）
    ├── 日常露出（真实使用场景）
    ├── 溯源 vlog（成分/工厂/配方）
    ├── 判断框架释放（买之前问这几个问题）
    └── 选品结果公示
```

带货内容围绕头发赛道的选品和品牌推进排期，另做专项规划。本方案先规划纯内容部分，再给出带货内容的营销框架。

### 系列一：「我在那边做过」

**定位**：第一大人设支柱，每周至少1条。

用行业内部视角，把普通消费者不知道、或者知道但没人敢说的事情说出来。不是揭秘为了揭秘，是帮用户建立自己的判断框架——知道了这些，以后花钱才能花对地方。

她在这个系列里的身份：从行业里走出来的内部人，不是站在消费者外面评价，是从里面说出来的。

#### 内容公式

```
反常识钩子（你以为……其实……）
→ 内部人拆解（在欧莱雅/大厂的时候，我们是这么做的）
→ 这件事背后的完整逻辑（成本/定价/营销/消费者心理）
→ 给一个拿来就能用的判断框架
→ 过来人狠话收束（笃定，不软着陆）
```

#### 为什么这个系列是核心

- **不可复制** —— 这些内容只有她能讲，因为她真的在那边做过。其他博主能讲成分科普，但讲不出"当初我们开会讨论的就是怎么让消费者觉得没有这个成分就输了"。
- **天然钩子** —— 反常识判断 + 行业内部信息，是最容易让人停下来看的内容类型。
- **信任积累** —— 每讲一次内部逻辑，用户对她的信任就加深一层。等到她推荐产品的时候，这些信任全部转化成转化率。
- **覆盖射程广** —— 美护全品类都在射程内，不局限于头发。但头发赛道的内容可以特别标记，持续深耕。

#### 选题库（20条）

**美护行业揭秘**
1. "一瓶300块的洗发水，成本到底是多少——我在大厂做过定价，告诉你一个数字"
2. "护肤品成分表里有这个成分，不代表有用——品牌方玩的是这个把戏"
3. "为什么网红爆款和真正好用的东西，几乎不重叠——流量逻辑和产品逻辑本来就是两件事"
4. "同一个工厂出来的东西，大牌卖500白牌卖50——我来告诉你哪些钱值得花"
5. "'无硅油更健康'这件事，是品牌讲的一个很漂亮的故事"
6. "我在欧莱雅管培生第一天，他们教了我什么——不是学产品，是学怎么看消费者心理"
7. "美妆品牌是怎么'制造'成分焦虑的——当年我们开会讨论的就是这件事"
8. "'经临床验证'这四个字，背后到底验证了什么"

**头皮头发深耕**
9. "脱发这件事，90%的人第一步就走错了——换洗发水是最没用的一步"
10. "为什么你换了十几款洗发水，头皮还是油——洗发水只是在处理结果，不是在解决原因"
11. "头皮是皮肤，但你护肤那套逻辑用在头皮上全错了"
12. "防脱成分哪些有用哪些是智商税——我把市面上的成分过一遍"
13. "你的头发细软，99%的人归因错了——和吃什么关系没那么大"
14. "头皮屑和头皮干，很多人分不清，但处理方法完全相反"

**消费判断框架**
15. "花了这么多年冤枉钱，我总结出一个买护肤品之前必须问自己的问题"
16. "凡是这么宣传的，基本上营销大于实效——我给你列三类"
17. "贵妇护肤品和平价平替，差距到底在哪——我从成本和配方给你拆一遍"
18. "医美这件事，什么值得做，什么是在收割焦虑——行业视角来说"
19. "为什么'大品牌'不等于'好产品'——品牌逻辑和产品逻辑不是同一件事"
20. "我做了20年美护，踩过最贵的三个坑——行业里的人也会被收割"

### 系列二：「这道题我答出来了」

**定位**：第二大人设支柱，每周1条。

中年女性真实在经历的那些困惑——外形、亲密关系、亲子、个人成长、财富——孙知羽用自己的真实经历做底，把这些问题一道道拆解清楚，给出笃定的判断和可执行的方法。

不是情感博主式的倾诉，也不是说教。是一个真实走过来的人，跟还在困惑里的人说说她是怎么想的。

#### 核心逻辑

这个系列的价值不是给答案，是帮人把问题想清楚。
她能做这件事，是因为她同时具备两样东西：清华训练出来的拆解方式，加上40年真实走过的人生——工薪家庭、清华、转行、大厂、婚姻、孩子、裸辞。这些经历让她和她的用户站在同一个地面上。

#### 内容公式

```
一个让人"说的就是我"的开场（场景/状态/困惑）
→ 我也走过这段路（真实的自我经历，允许不完美）
→ 后来我是怎么想明白的（拆解过程，用清华那套方法）
→ 给一个具体方法论（2-3步，可执行）
→ 笃定收束（"这是我的判断，你拿去用"，不是"希望对你有帮助"）
```

#### 这个系列的价值

- **覆盖射程最广** —— 不局限于美护，中年女性在关系、成长、财富上的所有困惑都在射程内。这是她从"美护博主"变成"中年女性精神家园"的核心内容。
- **情感连接最深** —— 用自己真实的走过的路做底，不是专家在讲道理，是过来人在说经历。这种内容的粘性远高于知识型内容。
- **差异化最强** —— 把清华思维工具 + 真实人生经历 + 中年女性处境三件事融在一起，是她独有的组合。
- **为长期信任打底** —— 这个系列不带货，只建信任。信任积累到一定程度，带货内容自然转化。

#### 选题库（20条）

**亲密关系**
1. "婚姻里最耗人的不是吵架，是那种'说了也没用'的感觉——我后来想明白这件事是怎么来的了"
2. "40岁了，我是怎么处理婚姻里的那些'说不清楚'的"
3. "普通家庭长大的人，在亲密关系里有一个特别容易踩的坑——我用了很多年才看见它"
4. "当初所有人都说那是'正确的选择'——为什么听别人的总是错的"
5. "我怎么处理和伴侣的'我以为你懂我'问题"

**亲子关系**
6. "和5岁孩子的硬冲突，我用这个方法处理——不是控制，是搞明白你在控制什么"
7. "我们这代妈妈最容易犯的一个错，是把自己的焦虑转嫁给孩子"
8. "要不要'鸡娃'这道题——我的判断标准就一个"
9. "工薪家庭长大的孩子，有些东西如果不主动看见，它会跟着你一辈子——我就是例子"
10. "我们给孩子留什么才算有用——不只是钱，是这件事"

**个人成长**
11. "40岁裸辞那天，我脑子里在想什么——不是因为勇敢，是因为这道账算清楚了"
12. "清华法学生去做美妆，我妈当时是什么反应——那是我第一次真正为自己做选择"
13. "40岁之后，我对'学习'这件事的认知彻底变了"
14. "从'别人家的孩子'到40岁裸辞——我是怎么一步步找到自己要的东西的"
15. "失败了、判断错了、走弯路了——我是怎么用清华那套方式复盘的"

**财富认知**
16. "中年女性对钱最常见的三个错误认知——我都犯过"
17. "工薪家庭出身，我是从什么时候开始不再为钱焦虑的——不是因为挣够了"
18. "消费降级之后，我发现真正值得花的钱是这些"
19. "为什么那么多女性对谈钱感到不舒服——这件事值得认真想"
20. "我花过的冤枉钱，不只是买了错的产品——更大的冤枉钱是在这里"

### 系列三：「清华教我这么想」

**定位**：差异化内容支柱，每两周1条。

这个系列是她和所有同类博主最本质的区别——她不只是给结论，她把思考过程交出来。
用清华训练出来的拆解方式，加上国内外前沿的研究和数据，把中年女性日常遇到的困惑从根上想清楚。每条内容里都有一个"拿走就能用的思维工具"。

这个系列不是炫学历。是真的在把工具给出去。

#### 内容公式

```
一个所有人都觉得"好像是这样"的默认认知（起手打破它）
→ 用清华那套方式重新拆解这件事（前提是什么，推导过程是什么，真正的结论是什么）
→ 引入国内外最新研究或数据（不是背资料，是"这对你意味着什么"）
→ 给一个具体思维工具（这件事你可以这么想 / 这三个问题你先问自己）
→ 一句笃定的话收束
```

#### 选题库（15条）

**思维工具类**
1. "同样一件事，普通人这么想，清华训练出来的人这么想——我用买洗发水举个例子"
2. "面对铺天盖地的信息，我怎么判断哪个是真的——三步筛选框架"
3. "大部分焦虑，本质上是同一件事——我把它拆开来给你看"
4. "为什么'努力'这件事没你想的那么重要——用对方向才是关键，方向怎么判断"
5. "决策的时候怎么判断这是'我真正想要的'还是'别人觉得好的'——清华给了我一把尺"

**健康与衰老认知**
6. "国外最新研究说，女性大脑在40岁前后某些维度反而在提升——但没人告诉你这意味着什么"
7. "压力是怎么让你的头发变少的——从激素到毛囊，我把这条链拆给你看"
8. "睡眠不够和头皮出油是同一个问题的两个结果——从数据出发来说这件事"
9. "为什么你运动了节食了但身材就是不变——中年女性代谢的真实逻辑"
10. "中年女性最容易忽视的三个身体信号——用数据而不是感觉来判断身体状态"

**关系与财富认知**
11. "为什么有些人越努力越累，有些人越活越轻松——两种人思维模式的本质区别"
12. "国外最新的行为经济学研究说，财富观念是怎么在家庭里代际传递的"
13. "亲子冲突的底层逻辑是什么——心理学给了一个答案，我用在了和我女儿的相处上"
14. "为什么中年女性普遍对谈钱感到不舒服——心理学和文化的双重拆解"
15. "清华那句话我记了20年，最近才真正用明白——'充分发挥自己的天赋和能力'"
